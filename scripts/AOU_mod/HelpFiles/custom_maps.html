About Custom Maps Overview This is a feature desined for the AOU Mod that enables you to change to any of the available custom maps you want without having to open up the planet map widndow. None of the actual custom maps come with the AOU Mod and so if any are missing an error message will pop up. Custom Map Download Links Latest versions of Onack's "Atlas of..." Map series (AoSL & AoRK) can be downloaded at (Note: the LoX maps have been intergrated into AoSL as of version 0.7): www.aomaps.com The latest version of CSP Map can be downloaded at: creativestudent.com The latest version of Spheremap can be downloaded at: www.ao-universe.com
